dom modifications.

To add javascript funtionality in your html/css code, add the tag script at the end of your html code, right before the body end tag.

You can choose to write your js code in the html document, or write it in a separate .js document. If you wanna do the latter, src your .js document, like so: 


                <script src = "script.js">

                </script>

We can access html using javascript by referencing it! If you do this: 


                document.body.innerHTML = "hacked."

This will access the entire body tag of html and wipe whatever you had off the face of the earth o_o

BUT if you wanna access something else, like the table of contents, you can do this: 


                var headingEl = document.getElementById("table");

                // since i set the id of the table of contents to be 'table'
                // to change the content in the thing :

                headingEl.innerHTML = "table of continents";

                // can put html tags in it 
                headingEl.innerHTML = "table of <em>continents</em>";
                // adds em to the word continents

Animating DOM with set interval: 


              // Step 1. What element do we want to animate?
              var countdown = document.getElementById("countdown");

              // Step 2. What function will change it each time?
              var countItDown = function() {
                countdown.textContent = parseFloat(countdown.textContent) - 1;  
              };
              
              // Step 3: Call that on an interval
              window.setInterval(countItDown, 1000);
Animating styles with requestAnimationFrame: 


              // making a picture/thing grow as time goes on 

              // Step 1. What element do we want to animate?
              var ohnoes = document.getElementById("ohnoes");
              ohnoes.style.width = "50px";
              
              // Step 2. What function will change it each time?
              var startTime = new Date().getTime();
              var makeItBigger = function() {
                  var currTime = new Date().getTime();
                  var newWidth = (50 + ((currTime - startTime)/1000) * 30);
                  ohnoes.style.width = newWidth + "px";  
                window.requestAnimationFrame(makeItBigger);
              };
              makeItBigger();

Using a JS library: Slideshow library 

 

            var slideShow = function(container) {
                this.images = [];
                this.curImage = 0;
                for (i = 0; i < container.childElementCount; i++) {
                    this.images.push(container.children[i]);
                    this.images[i].style.display = "none";
                }
                
                // Handle going to to the next slide
                var nextSlide = function() {
                    for (var i = 0; i < this.images.length; i++) {
                        this.images[i].style.display = "none";
                    }
                    this.images[this.curImage].style.display = "block";
                    this.curImage++;
                    if (this.curImage >= this.images.length) {
                        this.curImage = 0;
                    }
                    window.setTimeout(nextSlide.bind(this), 1000);
                };
                
                nextSlide.call(this);
            };
            slideShow(document.getElementById("slideshow"));
